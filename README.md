## Simple static site jump-off point.
Webpack + PostCSS

## Getting Started
  1. Install dependencies:
    `npm install`
  3. Run dev server:
    `npm run dev`
  4. Site running:
    `http://localhost:3000`
  5. Build for production:
    `npm run build`

## Contents
##### JS
  * none

##### PostCSS
  * Reset v2.0 (http://meyerweb.com/eric/tools/css/reset/)
  * postcss-import
  * postcss-comment
  * postcss-simple-vars
  * postcss-nested
  * autoprefixer

##### Components
