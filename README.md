## Simple static site jump-off point.
### Webpack + Sass
---

#### Getting Started
  1. Install dependencies
    ```
        npm install
    ```
  2. Run dev server
    ```
        npm run dev-server
    ```
  3. Build for production
    ```
        npm run build
    ```
---

#### JS
  * Jquery v3.1.1
